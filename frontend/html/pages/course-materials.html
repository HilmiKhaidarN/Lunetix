Modul 1: Pengenalan Machine Learning

Apa itu Machine Learning?

Machine Learning (ML) adalah cabang dari Artificial Intelligence (AI) yang memungkinkan komputer untuk belajar dari data tanpa diprogram secara eksplisit untuk setiap tugas. Alih-alih menulis aturan manual, kita memberikan data ke algoritma yang kemudian menemukan pola sendiri dan membuat prediksi.

Definisi Klasik

"Machine Learning adalah bidang studi yang memberikan komputer kemampuan untuk belajar tanpa diprogram secara eksplisit." - Arthur Samuel (1959)

Definisi Modern

"Sebuah program komputer dikatakan belajar dari pengalaman E terhadap tugas T dengan ukuran performa P, jika performanya pada T meningkat seiring pengalaman E." - Tom Mitchell (1997)

Mengapa ML Penting?

  • Volume data yang besar: Manusia tidak bisa memproses miliaran data poin secara manual.
  • Pola kompleks: Beberapa masalah terlalu kompleks untuk dirumuskan sebagai aturan eksplisit.
  • Adaptasi otomatis: Model ML dapat beradaptasi dengan data baru tanpa reprogramming ulang.
  • Personalisasi skala besar: Rekomendasi Netflix, Spotify, TikTok, dan e-commerce.
  • Otomasi keputusan: Deteksi fraud kartu kredit, diagnosis medis, filter spam email.

Perbedaan AI, ML, dan Deep Learning

Artificial Intelligence (AI)

Bidang paling luas — segala teknik yang membuat mesin "cerdas".

Machine Learning (ML)

Subset AI — mesin belajar dari data tanpa diprogram eksplisit.

Deep Learning (DL)

Subset ML — menggunakan neural network berlapis banyak.

Modul 2: Jenis-Jenis Machine Learning

1. Supervised Learning (Pembelajaran Terawasi)

Model dilatih menggunakan data berlabel — setiap input memiliki output yang benar.

Contoh Algoritma:

  • Regresi: Linear Regression, Polynomial Regression
  • Klasifikasi: Logistic Regression, Decision Tree, Random Forest, SVM

2. Unsupervised Learning (Pembelajaran Tidak Terawasi)

Model dilatih dengan data tanpa label. Model menemukan pola dan struktur tersembunyi sendiri.

Contoh Algoritma:

  • Clustering: K-Means, DBSCAN, Hierarchical Clustering
  • Dimensionality Reduction: PCA, t-SNE, UMAP

3. Reinforcement Learning (Pembelajaran Penguatan)

Agent belajar melalui trial-and-error dengan berinteraksi dengan lingkungan.

Contoh Aplikasi:

  • AlphaGo dan AlphaZero
  • Robotika dan autonomous driving
  • Optimasi strategi iklan

Modul 3: Algoritma Dasar ML

Linear Regression

Algoritma paling dasar untuk prediksi nilai kontinu. Mencari garis terbaik yang meminimalkan jumlah kuadrat error.

from sklearn.linear_model import LinearRegression
from sklearn.model_selection import train_test_split
from sklearn.metrics import mean_squared_error, r2_score
import numpy as np

# Data: ukuran rumah (m²) vs harga (juta rupiah)
X = np.array([[50],[75],[100],[125],[150],[80],[90],[110]])
y = np.array([300,450,600,750,900,480,540,660])

X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)

model = LinearRegression()
model.fit(X_train, y_train)
y_pred = model.predict(X_test)

print(f"R² Score: {r2_score(y_test, y_pred):.3f}")
print(f"RMSE: {mean_squared_error(y_test, y_pred, squared=False):.2f}")

Logistic Regression

Algoritma klasifikasi yang menggunakan fungsi sigmoid untuk menghasilkan probabilitas.

from sklearn.linear_model import LogisticRegression
from sklearn.datasets import load_breast_cancer
from sklearn.metrics import classification_report

data = load_breast_cancer()
X_train, X_test, y_train, y_test = train_test_split(
    data.data, data.target, test_size=0.2, random_state=42
)

model = LogisticRegression(max_iter=10000)
model.fit(X_train, y_train)
y_pred = model.predict(X_test)

print(classification_report(y_test, y_pred,
      target_names=['Malignant', 'Benign']))

Modul 4: Evaluasi Model

Confusion Matrix

Predicted Positive Predicted Negative
Actual Positive True Positive (TP) False Negative (FN)
Actual Negative False Positive (FP) True Negative (TN)

Metrik Evaluasi

  • Accuracy: (TP+TN)/Total
  • Precision: TP/(TP+FP)
  • Recall/Sensitivity: TP/(TP+FN)
  • F1-Score: 2×(P×R)/(P+R)
  • ROC-AUC: Area under ROC curve

Modul 5: Feature Engineering & Pipeline

Teknik Preprocessing

  • Handling Missing Values: Imputasi dengan mean/median/mode
  • Feature Scaling: StandardScaler atau MinMaxScaler
  • Encoding Categorical: One-Hot Encoding, Ordinal Encoding
  • Feature Selection: Correlation matrix, feature importance

Pipeline dengan Scikit-learn

from sklearn.pipeline import Pipeline
from sklearn.compose import ColumnTransformer
from sklearn.preprocessing import StandardScaler, OneHotEncoder
from sklearn.impute import SimpleImputer

# Definisikan kolom numerik dan kategorikal
numeric_features = ['age', 'income', 'score']
categorical_features = ['gender', 'city']

numeric_transformer = Pipeline([
    ('imputer', SimpleImputer(strategy='median')),
    ('scaler', StandardScaler())
])

categorical_transformer = Pipeline([
    ('imputer', SimpleImputer(strategy='most_frequent')),
    ('encoder', OneHotEncoder(handle_unknown='ignore'))
])

preprocessor = ColumnTransformer([
    ('num', numeric_transformer, numeric_features),
    ('cat', categorical_transformer, categorical_features)
])

# Full pipeline: preprocessing + model
full_pipeline = Pipeline([
    ('preprocessor', preprocessor),
    ('model', RandomForestClassifier(n_estimators=200))
])

full_pipeline.fit(X_train, y_train)
print(f"Test Accuracy: {full_pipeline.score(X_test, y_test):.3f}")

Modul 1: Neural Networks Fundamentals

Perceptron dan Activation Functions

Neural network terdiri dari neuron-neuron yang terhubung. Setiap neuron menerima input, melakukan komputasi, dan menghasilkan output melalui fungsi aktivasi.

Fungsi Aktivasi Populer:

  • Sigmoid: σ(z) = 1 / (1 + e⁻ᶻ)
  • ReLU: f(z) = max(0, z)
  • Tanh: tanh(z) = (eᶻ - e⁻ᶻ) / (eᶻ + e⁻ᶻ)
  • Softmax: Untuk multi-class classification

Modul 2: Backpropagation Algorithm

Chain Rule dalam Neural Networks

Backpropagation menggunakan chain rule dari kalkulus untuk menghitung gradient loss terhadap setiap weight dalam network.

∂L/∂w = ∂L/∂a × ∂a/∂z × ∂z/∂w

di mana L = loss, a = activation, z = weighted sum, w = weight

OpenCV Fundamentals

import cv2
import numpy as np

# Read image
img = cv2.imread('image.jpg')

# Convert to grayscale
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)

# Edge detection
edges = cv2.Canny(gray, 100, 200)

# Face detection
face_cascade = cv2.CascadeClassifier('haarcascade_frontalface_default.xml')
faces = face_cascade.detectMultiScale(gray, 1.1, 4)

# Draw rectangles around faces
for (x, y, w, h) in faces:
    cv2.rectangle(img, (x, y), (x+w, y+h), (255, 0, 0), 2)

# Show image
cv2.imshow('Detected Faces', img)
cv2.waitKey(0)
cv2.destroyAllWindows()

🎮 Reinforcement Learning

Advanced 11h 10m 900 students ⭐ 4.9

Modul 1: RL Fundamentals

Markov Decision Process (MDP)

MDP adalah framework matematis untuk modeling decision making. Terdiri dari:

  • State (S): Situasi saat ini
  • Action (A): Pilihan yang tersedia
  • Transition Probability (P): Probabilitas transisi state
  • Reward (R): Imbalan untuk setiap action
  • Discount Factor (γ): Faktor diskon untuk future rewards
